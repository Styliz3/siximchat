Getting Started

SimixChat API provides a simple yet powerful interface to access state-of-the-art language models. Here's how to get started:

JavaScript 
// Include the SimixChat API in your project
const simixchat = new SimixChatAPI({
  apiKey: 'your_api_key', // Optional, not required for free tier
  model: 'llama-3-70b-8192' // Default model
});
Python 
# Install the package
pip install simixchat

# Import and initialize
from simixchat import SimixChatAPI

simixchat = SimixChatAPI(
    api_key='your_api_key',  # Optional, not required for free tier
    model='llama-3-70b-8192'  # Default model
)

Basic Chat Completion

Generate responses from the model based on conversation history:

JavaScript 
async function chatWithAI() {
  const response = await simixchat.createChatCompletion({
    messages: [
      { role: 'system', content: 'You are a helpful assistant.' },
      { role: 'user', content: 'Tell me about quantum computing.' }
    ],
    model: 'llama-3-8b-8192', // Optional: override the default model
    max_tokens: 500,
    temperature: 0.7
  });

  console.log(response.choices[0].message.content);
}
Python 
async def chat_with_ai():
    response = await simixchat.create_chat_completion(
        messages=[
            {"role": "system", "content": "You are a helpful assistant."},
            {"role": "user", "content": "Tell me about quantum computing."}
        ],
        model="llama-3-8b-8192",  # Optional: override the default model
        max_tokens=500,
        temperature=0.7
    )
    
    print(response.choices[0].message.content)

Streaming Responses

Get real-time responses as they're generated for a better user experience:

JavaScript 
async function streamChatWithAI() {
  const stream = await simixchat.createChatCompletionStream({
    messages: [
      { role: 'system', content: 'You are a helpful assistant.' },
      { role: 'user', content: 'Write a short story about a robot.' }
    ],
    model: 'llama-3.1-8b-instant',
    stream: true
  });

  let fullResponse = '';
  
  for await (const chunk of stream) {
    const content = chunk.choices[0].delta.content || '';
    fullResponse += content;
    
    // Update UI with the incoming text
    console.log('Chunk received:', content);
  }
}
Python 
async def stream_chat_with_ai():
    stream = await simixchat.create_chat_completion(
        messages=[
            {"role": "system", "content": "You are a helpful assistant."},
            {"role": "user", "content": "Write a short story about a robot."}
        ],
        model="llama-3.1-8b-instant",
        stream=True
    )
    
    full_response = ""
    
    async for chunk in stream:
        content = chunk.choices[0].delta.content or ""
        full_response += content
        
        # Update UI with the incoming text
        print(content, end="", flush=True)

Frequently Asked Questions

Common questions about using SimixChat API

Is SimixChat API completely free to use?
Yes, SimixChat API is currently free to use for all supported models. We may introduce usage tiers in the future, but we're committed to maintaining a generous free tier.
Are there any rate limits?
While we aim to provide unrestricted access, we implement fair usage policies to ensure service quality for all users. If you anticipate high-volume usage, please contact us for custom solutions.
What's the difference between the available models?
The models vary in size (parameter count), specialization, and performance characteristics. Larger models like Llama-3-70B offer more capabilities but may have higher latency, while specialized models like Qwen-2.5-Coder excel at specific tasks like code generation.
Can I use the API for commercial applications?
Yes, you can use SimixChat API in commercial applications. However, please review our Terms of Service for full details on allowed usage.
How do I report issues or get support?
For technical support or to report issues, please contact us through our support email or visit our GitHub repository to file an issue ticket.
Can I fine-tune models for my specific use case?
Currently, model fine-tuning is not supported in the free tier. Contact us for information about custom solutions if you need specialized model training.
